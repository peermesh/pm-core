Get started by enabling a profile in your docker-compose configuration.

1

Enable Profiles

Add profiles to your .env file:

COMPOSE_PROFILES=postgresql,redis
2

Start Services

Run docker-compose to start your services:

docker compose up -d
3

Monitor Your Stack

This dashboard will automatically display your running modules and their resource usage.

Available Profiles

postgresql Database
redis Cache
monitoring Metrics
backup Data
identity Auth
secrets Vault

Quick Commands

make validate Validate compose config
docker compose ps List running containers
docker compose logs -f Follow container logs
