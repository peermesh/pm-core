Add Remote Instance

Register another PeerMeshCore dashboard running on a different server. The remote instance must be network-accessible from this dashboard's container.

A human-readable label for this remote server (e.g., "Production VPS", "Staging Server")

The full HTTPS URL where the remote PeerMeshCore dashboard is accessible (e.g., https://staging.peermeshcore.example.com)
